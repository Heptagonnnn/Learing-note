# XML
[toc]


##定义

XML---- Extensible MarkupLanguage

可扩展标记语言



## 用途

1. 存储数据

2. 配置文件


## 基本语法


### 文档生声明

1. 必须位于文件第一行

2. 用于声明版本等信息

```XML
<?xml version="1.0" encoding="UTF-8"?>
```
version: xml版本

encoding: 字符集

M
### 规则
1. 所有xml元素必须闭合

2. 大小写敏感

3. 正确嵌套

4. 文档必须有跟元素

5. 元素的属性值必须加引号

6. 实体引用

    在xml中，一些字符拥有特殊的意义，如果把特殊字符放在xml元素中，会发生错误

    因为规定了5个预定义的实体引用，用于代替实体符号

    |实体符号|实体引用|
    |----|----|
    |<|\&lt;|
    |>|\&gt;|
    |&|\&amp;|
    |'|\&apos;|
    |"|\&quot;|
